# plan-a-trip-to-mars

## Install

With `poetry`:

```sh
poetry install
```

and run with:

```sh
poetry run plan-a-trip-to-mars
```

Without `poetry` (using `pip`):

```sh
python -m pip install -e .
```

and run with:

```sh
plan-a-trip-to-mars
```
